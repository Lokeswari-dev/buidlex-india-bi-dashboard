# Buildex India - Sales & Supply Chain Analytics Dashboard

An interactive real-time dashboard built using **Python** and **Streamlit** to analyze construction material sales, payment risks, and regional performance for Buildex India.

## 🚀 Features
- **KPI Metrics:** Track Total Revenue, Total Orders, and Pending Collections in Millions (M).
- **Payment Risk Analysis:** Stacked bar charts to identify 'Pending' vs 'Paid' status by material.
- **Regional Insights:** City-wise sales performance across Hyderabad, Pune, Vizag, etc.
- **Interactive Filters:** Filter data by Location and Material Name instantly.
- **Data Audit:** View raw transaction data for detailed verification.

## 🛠️ Tech Stack
- **Language:** Python
- **Dashboard:** Streamlit
- **Visualization:** Plotly Express
- **Data Handling:** Pandas & NumPy
